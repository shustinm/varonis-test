Normalizer
----------

Install dependencies:
```bash
pip install -r requirements.txt
```

Test:
```bash
python -m pytest
```

Regular run:
```bash
python main.py
```

Serverless WSGI server (how ironic):
```bash
serverless wsgi serve
```

Serverless offline run:
```bash
serverless offline
```
